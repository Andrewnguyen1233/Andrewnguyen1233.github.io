The Story Of My Game

At first I was thinking of making a geometry dash, but then Mario looked better so I dropped geometry dash and went for Mario, then I realized that I didn't know how to make the screen look like it is moving, so thought of a game that does not require the screen to move. That's when the game knife hit popped in my head, I tried to make the the game as close to the original as possible, but it seemed impossible to get the sprites to do what I want it to do. So i kept the idea of knife hit in mind, but changed it up so that I could have more control over what happens, that's when I created "Target Hit"

Scoring System & Randomness Occurance

When the kinfe hits the target the user gets one point. The randomness occurs when you reach 15 points or higher because the game has a chance of spawning mroe tiles.

Difficulty Increase and End Game

After the first 15 knives hit the target the horizontal line starts moving faster making you have to be more cautious on when to click. The game ends only when the knife hits the moving line other than that, it will keep getting faster and faster make it harder to play.

Relection of Game

I made this game because I used to play it a lot and I think I got to level 200 something, which inspired me to try and create an easy game that goes on forever.
